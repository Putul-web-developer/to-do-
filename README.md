# Task Manager
This is a simple Task Manager project.
## Description
In this project, users can manage their daily tasks easily.

## Features
- User Registration
- User Login
- Dashboard page after login
- Add new tasks
- Edit tasks
- Delete tasks
- Logout
- Saved tasks can be checked anytime

## How it works
1. User first registers an account  
2. Then user logs in  
3. After login, user is redirected to dashboard  
4. User can add tasks  
5. User can edit or delete tasks  
6. User can logout  
7. Tasks are saved and can be viewed anytime
